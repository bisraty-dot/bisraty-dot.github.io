NYC 311 DASH

The NYC 311 Call Activity Dashboard is an interactive data visualization project created using Bokeh, aimed at analyzing non-emergency call activity in New York City from 2020 onwards. Utilizing the 311 Service Requests dataset from NYC Open Data, this dashboard allows users to explore call volume trends across the five boroughsâ€”Bronx, Manhattan, Brooklyn, Staten Island, and Queens. Key features include visualizations of call volumes by location, category, time, and type, enhanced with interactivity such as date sliders for animating data over time and tooltips for detailed insights. This project, which required setting up Git, Google Developer credentials, and a Socrata API token, demonstrates my proficiency in data cleaning and analysis with Python libraries like Pandas and Numpy, and my ability to create dynamic and informative visualizations using Bokeh. The dashboard serves as a powerful tool for understanding 311 call trends, aiding in resource allocation and case prioritization for city planners and analysts.
