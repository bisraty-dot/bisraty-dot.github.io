## 💼 Projects

### 🔹 Sales Analysis Dashboard | Power BI  
A dynamic, real-time Power BI dashboard that enhances sales reporting and performance monitoring.

- Tracks internet sales by customer and product
- Daily updated overview with budget comparisons
- Allows interactive filtering by region, product, customer, and sales rep
- Built to streamline sales follow-ups and empower reps with actionable insights

🔗 *[View Project](https://bisraty-dot.github.io/#)*

---

### 🔹 NYC 311 Call Activity Dashboard | Python + Bokeh  
An interactive dashboard analyzing NYC's non-emergency 311 calls from 2020 onward.

- Visualizes call volume by borough, category, and time
- Built with Pandas, and NumPy
- Integrated with the NYC Open Data API using Socrata and Google Developer setup
- Includes dynamic sliders and tooltips for rich, temporal exploration

🔗 *[View Project](https://bisraty-dot.github.io/#)*

---

### 🔹 Global Coffee Market Analysis | Tableau + Python  
An in-depth analysis of the global coffee market (2010–2020), inspired by Ethiopia’s coffee heritage.

- Explores agricultural land per capita, production efficiency, and research investment
- Combines Tableau dashboards with Excel preprocessing and Python analysis
- Sources include Our World in Data and the World Bank
- Addresses data quality challenges and global economic disparities

🔗 *[View Project](https://bisraty-dot.github.io/#)*
